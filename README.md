### 08- Date of birth [Solution]()

##### Challenge Description

Write a function that receives a date and time in the form of a string. The function checks whether the entered date is a valid age date or not. So the function returns the value True if the entered date is a valid date, and returns the value False if the entered date is incorrect, according to the following conditions:

1. The entered date must be before the current date

2. The entered date must be a minimum of 4 years before today's date

3. The entered date must not exceed 200 years in the past from the current date
